📡

Extended Kalman Filter — polar radar

Native polar measurement model h(x) = [√(x²+y²), atan2(y,x)] with analytic 2×4 Jacobian. Avoids the Cartesian approximation error that grows as r·σ_θ at long range. Angle-normalised innovation prevents ±π wraparound blowups.

🎯

Mahalanobis Gating

Innovations tested against a χ²(2 DOF, 99%) gate before Hungarian assignment. Infeasible pairs get sentinel cost 10⁹ — clutter is rejected at the gate, not after the assignment. Tuning gate tightness trades missed detections vs. clutter ingestion.

🔀

GNN Data Association

Global Nearest-Neighbour via scipy.optimize.linear_sum_assignment (Jonker-Volgenant, O(n³)). One-to-one globally optimal assignment — order-independent, unlike greedy nearest-neighbour which is sequence-dependent and can cascade ID switches.

🛤️

Track Lifecycle

New measurement → TENTATIVE (zero velocity, large Pᵥ). Two consecutive hits → CONFIRMED. Three consecutive misses → DELETED. Stale tentative tracks pruned after 3 frames — eliminates single-blip false alarms without multi-hypothesis tracking.

Class Radar model R matrix When to prefer
KFTracker Convert polar → Cartesian first, linear H Fixed isotropic diag[4, 4] m² approx Short range, near-boresight, quick prototype
